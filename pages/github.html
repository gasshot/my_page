• GitHub는 내가 만든 코드를 저장하고 관리하는 서비스입니다.
  • GitHub Pages를 사용하면 HTML, CSS, JavaScript로 만든 웹 페이지를 무료로 배포할 수 있습니다.

GitHub에서 하는 일

  • 프로젝트 파일을 저장합니다.
  • 코드 변경 기록을 관리합니다.
  • GitHub Pages로 웹 사이트를 배포합니다.

배포 기본 순서

  1. GitHub 저장소를 생성합니다.
  2. 프로젝트 파일을 업로드합니다.
  3. Settings 메뉴로 이동합니다.
  4. Pages 메뉴에서 배포 브랜치를 선택합니다.
  5. 생성된 주소로 접속합니다.

Git 기본 명령어 흐름

Git을 처음 설정하고, 프로젝트를 GitHub 저장소에 업로드할 때 사용하는 기본 명령어입니다.

Git Bash 
# Git 사용자 정보 설정
git config --global user.name "강인오"
git config --global user.email "your@email.com"
git config --global core.editor "code --wait"

# 새 프로젝트 시작
mkdir my-project
cd my-project
git init

# 파일 추가 및 첫 커밋
echo "# My Project" > README.md
git add README.md
git commit -m "feat: 프로젝트 초기 설정"

# GitHub 원격 저장소 연결
git remote add origin https://github.com/username/my-project.git
git branch -M main
git push -u origin main
