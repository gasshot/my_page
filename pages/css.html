CSS

CSS는 HTML 요소의 디자인을 담당합니다. 색상, 크기, 여백, 배치, 애니메이션 등을 설정할 수 있습니다.

CSS의 기본 역할

  • 글자 색상과 크기를 변경합니다.
  • 요소의 위치와 간격을 조정합니다.
  • 반응형 웹 디자인을 만들 수 있습니다.

기본 예제

.card {
    width: 300px;
    padding: 20px;
    background-color: white;
    border-radius: 10px;
    box-shadow: 0 6px 16px rgba(0, 0, 0, 0.12);
}

.card:hover {
    transform: translateY(-6px);
}
