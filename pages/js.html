JavaScript

JavaScript는 웹 페이지에 동작을 추가하는 언어입니다. 버튼 클릭, 이미지 변경, 메뉴 열기, 데이터 처리 등을 구현할 수 있습니다.

JavaScript의 기본 역할

  • 사용자의 클릭이나 입력에 반응합니다.
  • HTML 내용을 동적으로 변경합니다.
  • 조건문과 반복문으로 기능을 처리합니다.

기본 예제

const button = document.querySelector(".btn");
const title = document.querySelector(".title");

button.addEventListener("click", function () {
    title.innerText = "버튼을 클릭했습니다.";
});
