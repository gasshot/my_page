• HTML은 웹 페이지의 뼈대를 구성하는 마크업 언어입니다.
  • 시맨틱 태그를 사용하면 코드의 의미가 더 명확해집니다.

기본 문서 구조

모든 HTML 문서는 기본 구조를 가집니다.

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>페이지 제목</title>
</head>
<body>
    <h1>안녕하세요</h1>
</body>
</html>

시맨틱 태그

태그 설명
<header> 페이지나 섹션의 머리글
<nav> 메뉴 영역
<main> 주요 콘텐츠 영역
<footer> 바닥글 영역
